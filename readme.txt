Included folders and files are:
- article_figures: all figures in the manuscript (.png format)
- RF_Pulse_sequences: Contains all the sequences used in the experiment, developed in the Pulseq programming environment
- matlab_functions: Matlab functions used for creating the RF waveforms (T1rho pulse schemes) and for data analysis
- README.txt: this file describing the contents of the dataset
